# 摊位装饰页面重构计划（修订版7）

## 目标
根据价格机制重构摊位装饰的介绍页面，大幅增加装饰种类，降低单个装饰的收益，让玩家通过组合搭配来堆叠属性。解锁条件改为消耗不同时期对应的凭证升级获得称号后解锁，各时期凭证消耗总量保持相对均衡。

## 设计原则
1. **高投资高回报**：保持价格跳跃，后期装饰价格更高但收益也更高
2. **放置类游戏特性**：考虑后期数值膨胀，设计更多可堆叠的装饰
3. **策略性搭配**：通过不同类型装饰的组合，实现个性化配置
4. **国风命名**：使用符合西游/古代商业风格的等级名称
5. **丰富选择**：提供大量装饰供玩家收集和搭配
6. **凭证升级**：通过消耗不同时期对应凭证获得称号解锁装饰购买权限
7. **总量均衡**：各时期凭证消耗总量差距不要过大

## 重构方案

### 1. 装饰数量扩展
从6个扩展到36个装饰，分为6个等级，每个等级6个装饰：
- 草创期（6个）：消耗1000草创凭证获得"小商贩"称号解锁
- 立业期（6个）：消耗1500立业凭证获得"行商"称号解锁
- 兴业期（6个）：消耗2000兴业凭证获得"富商"称号解锁
- 旺业期（6个）：消耗2500旺业凭证获得"巨贾"称号解锁
- 鸿业期（6个）：消耗3000鸿业凭证获得"商业大亨"称号解锁
- 传世期（6个）：消耗4000传世凭证获得"大富翁"称号解锁

### 2. 单个装饰收益调整
大幅降低单个装饰收益，鼓励玩家购买大量装饰进行组合：
- 草创期：每项+1%-2%成交率
- 立业期：每项+2%成交率
- 兴业期：每项+2%-3%成交率，开始引入离线收益
- 旺业期：每项+3%成交率，离线收益+2%-3%
- 鸿业期：每项+3%-4%成交率，离线收益+3%-4%，税费减免
- 传世期：每项+4%-5%成交率，离线收益+4%-5%，特殊效果

### 3. 价格体系（保持跳跃性）
- 草创期：30铜 - 200铜
- 立业期：300铜 - 2银
- 兴业期：3银 - 20银
- 旺业期：30银 - 200银
- 鸿业期：300银 - 2000银（2金）
- 传世期：3000银（3金） - 20000银（20金）

### 4. 装饰分类（增加策略性）
- **招牌类**：主打成交率提升（8个）
- **陈列类**：成交率+离线收益平衡（8个）
- **氛围类**：时段/特殊条件加成（8个）
- **风水类**：税费减免+综合属性（6个）
- **神像类**：顶级综合效果（6个）

## 页面内容结构

### 1. 装饰系统概述
- 装饰的作用：提升成交率、增加离线收益、减免税费
- 叠加规则：所有已购买装饰的效果累加
- 跨摊位通用：购买后所有摊位共享
- 收集乐趣：36个装饰供玩家逐步解锁收集

### 2. 解锁机制说明
- 需要消耗对应时期的凭证获得称号才能解锁该等级装饰
- 不同时期需要不同类型的凭证
- 凭证通过交易、任务、成就等途径获得
- 获得称号后该称号的属性永久生效
- 装备称号后即可购买该等级的所有装饰

### 3. 凭证获取指南
- 草创凭证：新手期主要通过基础交易和任务获得，每笔交易5-10个
- 立业凭证：通过日常经营任务积累，每笔交易3-6个
- 兴业凭证：参与跨区域贸易和商会贡献，每笔交易2-4个
- 旺业凭证：大额交易和商会任务，每笔交易1-3个
- 鸿业凭证：传奇任务和限时活动，每笔交易1-2个
- 传世凭证：巅峰成就和赛季奖励，每笔交易1个

### 4. 装饰类型说明
| 类型 | 数量 | 特点 | 适合场景 |
|------|------|------|----------|
| 招牌类 | 8个 | 成交率加成 | 追求快速成交 |
| 陈列类 | 8个 | 成交率+离线收益平衡 | 综合发展 |
| 氛围类 | 8个 | 特殊时段加成 | 特定时段经营 |
| 风水类 | 6个 | 税费减免+属性 | 大额交易玩家 |
| 神像类 | 6个 | 顶级综合效果 | 后期毕业配置 |
