摆摊系统框架搭建指南

摆摊系统完全指南 | 开发者框架参考 | 搭建模拟器 | 文档版本：v1.0

玩家快速入门

一句话理解摆摊系统：摆摊系统是《蛙噻！好摊》的核心玩法。你需要通过低买高卖来积累财富，在四条街道的800+摊位中选择合适位置，上架商品并设定价格，通过观察市场反馈来调整策略，最终成为一代商业传奇。

选择摊位

在四条街道中选择一个空摊位，位置越好人流量越大

上架商品

从背包中选择商品上架，设定一个有竞争力的价格

观察反馈

NPC议价意味着定价偏高，快速成交说明定价合理

使用模拟器

切换到"搭建模拟器"标签，测试不同场景的效果

摆摊基础流程

第一步：选择位置

集市由四条街道组成，每条街道有200+摊位，各有特色：

西街道具炼金师必去药水、卷轴
北街装备战士天堂武器、防具

交易税费5%成交金额的5%
收摊费用0收摊不收取任何费用

第三步：观察反馈调整策略

反馈类型 含义 建议操作
快速成交（15分钟内） 定价偏低或市场短缺 下次可适当提价10-20%
正常成交（1-2小时） 定价合理 保持当前定价策略
收到NPC议价 定价偏高 参考议价价格调整，或接受议价
长时间无人问津 定价严重过高 收摊后重新开摊定价

摊位自动前移机制

当某个摊位收摊后，该摊位后方的所有摊位会自动向前移动填补空位。只要你持续经营的时间够久，摊位会慢慢移动到更靠前的位置。

常见问题

出摊后可以修改商品价格吗？ -

不可以。出摊后无法更新上架的商品，也无法修改价格。如果需要更换商品或调整价格，必须先收摊（已售商品收益会结算，未售商品返还背包），然后重新选择摊位出摊。

摊位自动前移是怎么运作的？ -

当某个摊位收摊后，该摊位后方的所有摊位会自动向前移动填补空位。摊位始终向街道中心方向（左侧）靠拢，保持紧凑排列。这个机制鼓励玩家长期经营，即使一开始只能在边缘位置出摊，只要坚持经营，终有机会移动到黄金位置。

离线时摊位还在营业吗？ -

是的。《蛙噻！好摊》是放置类游戏，摊位支持离线自动运营。重新上线时自动结算离线期间的收益，离线时间最长累计24小时。离线收益受摊位位置和装饰加成影响。

如何解锁更多装饰？ -

装饰通过消耗不同时期的凭证来获得称号解锁：消耗1000草创凭证获得"小商贩"称号，消耗1500立业凭证获得"行商"称号，以此类推。不同时期需要不同类型的凭证，凭证通过交易、任务、成就等途径获得。

系统架构

摆摊系统采用模块化架构设计，主要包含以下核心模块：

  • StallManager（摊位管理模块）：负责摊位的选择、占用、撤离、自动对齐
  • TradingSystem（交易系统模块）：负责商品的上架、下架、议价、成交
  • PriceSystem（价格系统模块）：负责市场价格计算、定价反馈
  • DecorationSystem（装饰系统模块）：负责装饰解锁、称号加成、效果计算

核心算法实现

出摊流程

startStall 实现代码 - 
// 出摊流程
async function startStall(stallId, items, player) {
    const stall = stallManager.selectStall(stallId);
    if (!stall) return { success: false, message: "摊位不存在" };

    // 1. 计算手续费
    let totalFee = 0;
    items.forEach(listing => {
        const marketPrice = priceSystem.calculateMarketPrice(listing.itemId, stall.position.street);
        totalFee += marketPrice * 0.01; // 1% 上架费
    });

    // 2. 检查余额
    if (player.money < totalFee) {
        return { success: false, message: "余额不足，无法出摊" };
    }

    // 3. 扣除费用并上架商品
    player.money -= totalFee;
    items.forEach(listing => {
        tradingSystem.listItem(listing.itemId, listing.price, listing.quantity);
    });

    // 4. 更新摊位状态
    stallManager.occupyStall(stallId, player.id);

    return { success: true, message: "出摊成功", fee: totalFee };
}

收摊流程

endStall 实现代码 - 
// 收摊流程
async function endStall(stallId, player) {
    const stall = stallManager.selectStall(stallId);
    if (!stall || stall.owner !== player.id) {
        return { success: false, message: "摊位不属于你" };
    }

    // 1. 获取已售商品收益
    const soldItems = tradingSystem.getSoldItems(stallId);
    let totalEarnings = 0;
    soldItems.forEach(sale => {
        const tax = sale.price * 0.05; // 5% 交易税
        totalEarnings += sale.price - tax;
    });

    // 2. 获取未售出商品
    const unsoldItems = tradingSystem.getUnlistedItems(stallId);
    unsoldItems.forEach(item => {
        player.addToInventory(item);
    });

    // 3. 撤离摊位
    stallManager.vacateStall(stallId);

    // 4. 返回收益
    player.money += totalEarnings;

    return {
        success: true,
        earnings: totalEarnings,
        unsoldCount: unsoldItems.length
    };
}

离线收益计算

calculateOfflineEarnings 实现代码 - 
// 离线收益计算
function calculateOfflineEarnings(offlineTime, stall) {
    const maxOfflineTime = 24 * 60 * 60 * 1000; // 24小时上限
    const effectiveTime = Math.min(offlineTime, maxOfflineTime);

    // 基础收益
    let baseEarnings = 0;
    stall.listedItems.forEach(listing => {
        const avgSellTime = calculateAvgSellTime(listing);
        const expectedSales = effectiveTime / avgSellTime;
        baseEarnings += listing.price * expectedSales * 0.95;
    });

    // 位置加成
    const positionBonus = {
        "center": 1.5,
        "middle": 1.25,
        "edge": 1.0
    }[stall.position.type];

    // 装饰加成
    const decorationBonus = decorationSystem.getTotalBonus();

    // 最终收益
    const finalEarnings = baseEarnings
        * positionBonus
        * (1 + decorationBonus.offlineRate)
        * (1 - decorationBonus.taxReduction);

    return Math.floor(finalEarnings);
}

摊位自动对齐

autoAlignStalls 实现代码 - 
// 摊位自动前移对齐
autoAlignStalls(street) {
    // 当有摊位收摊后，后面的摊位自动前移
    const streetStalls = this.stalls.filter(s => s.position.street === street);
    streetStalls.sort((a, b) => a.position.index - b.position.index);

    let currentIndex = 0;
    streetStalls.forEach(stall => {
        if (stall.status === "occupied") {
            stall.position.index = currentIndex++;
        }
    });
}

完整类实现

Game主类

完整代码 - 
class Game {
    constructor() {
        this.player = { ...PLAYER_DATA };
        this.stalls = this.initStalls();
        this.currentStall = null;
    }

    initStalls() {
        const stalls = [];
        let id = 1;
        for (const street of Object.keys(STALL_POSITIONS)) {
            for (let i = 0; i < 200; i++) {
                stalls.push({
                    id: `stall_${id++}`,
                    street: street,
                    index: i,
                    owner: null,
                    listedItems: []
                });
            }
        }
        return stalls;
    }

    selectStall(stallId) {
        this.currentStall = this.stalls.find(s => s.id === stallId);
        return this.currentStall;
    }

    startStall(stallId, items) {
        const stall = this.selectStall(stallId);
        if (!stall || stall.owner) {
            return { success: false, message: "摊位不可用" };
        }
        let fee = 0;
        items.forEach(item => { fee += item.price * 0.01; });
        if (this.player.money < fee) {
            return { success: false, message: "余额不足" };
        }
        this.player.money -= fee;
        stall.owner = this.player.id;
        stall.listedItems = items;
        return { success: true, message: "出摊成功", fee: fee };
    }

    endStall(stallId) {
        const stall = this.stalls.find(s => s.id === stallId);
        if (!stall || stall.owner !== this.player.id) {
            return { success: false, message: "摊位不属于你" };
        }
        let earnings = 0;
        stall.listedItems.forEach(item => {
            earnings += item.price * 0.95; // 5%税费
        });
        const unsold = [...stall.listedItems];
        stall.listedItems = [];
        stall.owner = null;
        this.player.money += earnings;
        return { success: true, earnings: earnings, unsold: unsold };
    }
}
